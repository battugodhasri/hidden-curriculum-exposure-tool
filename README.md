# Hidden Curriculum Exposure Tool

## Project Overview
Hidden Curriculum Exposure Tool is a Flask-based web application developed to explore and analyze hidden learning opportunities, skills, and knowledge areas that are often not covered in traditional education.
The application provides an interactive platform to help users understand additional skills required for personal and professional development.

## Live Demo
https://hidden-curriculum-exposure-tool.onrender.com

## Features
- Flask-based web application
- Interactive user interface
- Dynamic web page rendering
- Data-driven analysis
- Structured data management
- Deployed as a live application

## Installation and Setup

### Clone the Repository
```bash
git clone https://github.com/battugodhasri/hidden-curriculum-exposure-tool.git
```

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run the Application
```bash
python app.py
```

The application will be available at:

```
http://127.0.0.1:5000
```

## Deployment
The application is deployed using Render.

Build Command:

```
pip install -r requirements.txt
```

Start Command:

```
python app.py
```
